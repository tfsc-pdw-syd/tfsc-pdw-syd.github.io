# TFSC Special Issue Paper Development Workshop 2026

This is a small informational site for the TFSC Special Issue Paper Development Workshop hosted with AI-EEKE 2026.

Files
- [index.html](index.html) - main page
- [styles.css](styles.css) - stylesheet

Assets

Place the header logos in an `assets/` folder at the project root with these filenames:

- `assets/uts-logo.png` - University of Technology Sydney logo (suggested size: 360x100)
- `assets/tfsc-logo.png` - TFSC logo (suggested size: 360x100)


Organisers

- University of Technology Sydney (UTS) - hosting the paper development workshop
- In collaboration with Technological Forecasting & Social Change (TFSC)

Local preview

Run a simple local web server from the project folder. Examples:

```bash
python -m http.server 8000
# then open http://localhost:8000 in your browser
```

Or with Node (if you have `http-server`):

```bash
npx http-server -c-1
```

Hosting

- GitHub Pages: push to a repo and enable Pages from the `main` branch (or `gh-pages`).
- Netlify / Vercel: drag-and-drop the site folder or connect your repo.
